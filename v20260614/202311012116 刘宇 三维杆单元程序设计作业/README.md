\# 三维杆单元程序运行说明

1\. 运行环境

&#x20;  - Python 3.7+

&#x20;  - 依赖库：numpy

&#x20;  安装依赖：pip install numpy



2\. 运行方式

&#x20;  直接执行代码：python truss3d.py



3\. 程序功能

&#x20;  1. 计算三维杆单元长度、方向余弦

&#x20;  2. 生成全局坐标系6×6单元刚度矩阵

&#x20;  3. 由节点位移求解应变、应力、轴力

&#x20;  4. 节点重合异常检测

&#x20;  5. 内置两个标准算例自动验证

&#x20;  6. 刚度矩阵对称性、奇异性、刚体位移、物理意义验证



4\. 单位说明

&#x20;  - 长度：m

&#x20;  - 弹性模量：Pa

&#x20;  - 面积：m²

&#x20;  - 位移：m

&#x20;  - 应力：输出转换为 MPa

&#x20;  - 轴力：N
